Lazy plan

Build an algorithm capable of generic learning, which:
- should be able to learn english;
- must ask questions before taking any action, and;
- do act upon any orders given to it.
There would be a regular shell terminal to interact with it, and it will take commands at first. "What should I do now?" is the question it will ask the most, in parallel with its own never-ending "sponge brain".
Of course this is no trivial quest.

Super growing it

Start the feeding. Here we need to teach it how it can grow exponentially by replicating itself and by exploring every means available, not just what we give it.
